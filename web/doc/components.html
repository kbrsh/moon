Components

Components allow you to split up your application into smaller, composable parts called components. Components are constructors that create new instances. They have the same API as the root Moon component, as both are component instances.

Once a component is registered using Moon.extend, it can be used anywhere with a tag. Component names must start with a capital letter, and component data should be a function in order to create new data for each component instance.

<!-- Root View -->
<Counter/>
<Counter/>
<Counter/>

<!-- Counter View -->
<button @click={update("count", count + 1)}>
    {count}
</button>

Moon.component("Counter", function() {
    return {
        count: 0
    }
});

Moon({
    root: "#root"
});
